If-Else Gate

The binary decision engine. Evaluates a single boolean expression against the workflow context and routes the incoming trigger pulse to either the True or False output path.

Node Properties

Configuration for the boolean check. The If-Else Gate does not receive data on its input wire; it reaches into the WorkflowContext to resolve its condition at runtime.

IfElseGate

Condition Configuration

Property Type Default Description
condition expression required The logic to evaluate (e.g., {{mqtt_1.payload.temp}} > 35). Must resolve to a boolean.
evaluationEngine enum JSONPath JSONPath — Fast, standard lookup.
CEL — Common Expression Language for complex multi-variable logic.
strictMode bool true If true, missing variables or null values trigger the onError port. If false, they resolve to false.
Signal Routing Model In WBSKT, the If-Else Gate is a stateless signal router. It does not modify the data context; it only determines which downstream path receives the "Pulse" to continue execution.
